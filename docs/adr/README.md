# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for DocSynth. ADRs document significant technical decisions made during the project's development.

## Index

| ADR                                     | Title                   | Status   | Date       |
| --------------------------------------- | ----------------------- | -------- | ---------- |
| [0001](0001-core-technology-choices.md) | Core Technology Choices | Accepted | 2026-02-09 |

## What is an ADR?

An ADR records a significant architectural decision along with its context and consequences. They help future contributors understand **why** things were built a certain way.

## Creating a New ADR

1. Copy the template: `cp docs/adr/0001-core-technology-choices.md docs/adr/NNNN-title.md`
2. Fill in the Context, Decision, and Consequences sections
3. Add the entry to the index table above
4. Submit as part of a PR
